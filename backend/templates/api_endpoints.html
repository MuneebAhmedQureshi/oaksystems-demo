Process Automation Feasibility and Prioritization REST API v1.0.0

üîê Authentication Required

Most endpoints require JWT authentication. Include the token in the Authorization header: Bearer <token>

Authentication Endpoints

User Authentication & Management

Handle user registration, login, and profile management

POST /api/auth/register/
Register a new user account
POST /api/auth/login/
Authenticate user and get JWT tokens
GET /api/auth/profile/
Get current user's profile information
PUT /api/auth/profile/update/
Update user profile information
GET /api/auth/users/
List all users (admin only)

Process Assessment & Reports

Process Assessments

Create, manage, and evaluate Process Automation Feasibility and Prioritizations

GET /api/tasks/assessments/
List all process assessments with filtering and search
POST /api/tasks/assessments/
Create a new process assessment
GET /api/tasks/assessments/{id}/
Get detailed information about a specific assessment
PUT /api/tasks/assessments/{id}/
Update an existing assessment
DELETE /api/tasks/assessments/{id}/
Delete a process assessment
POST /api/tasks/assessments/bulk/
Create multiple assessments at once

Reports

Generate and manage comprehensive assessment reports

GET /api/tasks/reports/
List all generated reports
POST /api/tasks/reports/
Create a new report from selected assessments
GET /api/tasks/reports/{id}/
Get detailed report information
POST /api/tasks/reports/{id}/ai-conclusion/
Generate AI-powered conclusion for a report
GET /api/tasks/reports/{id}/download/csv/
Download report as CSV file
GET /api/tasks/reports/{id}/download/pdf/
Download report as PDF file

Dashboard

Get statistical data and analytics for dashboard views

GET /api/tasks/dashboard/stats/
Get dashboard statistics and summary data

Categories

Manage process categories for organization

GET /api/tasks/categories/
List all process categories
POST /api/tasks/categories/
Create a new process category

AI Features

Machine Learning Analysis

Advanced AI-powered analysis and predictions for process automation

POST /api/ai/similarity-analysis/
Perform K-means clustering to find similar processes
POST /api/ai/predict-success/
Predict automation success probability for a process
POST /api/ai/optimization-suggestions/
Generate AI-powered optimization recommendations
GET /api/ai/analysis-history/
Get history of all AI analyses performed

Automation Templates

Templates & Recommendations

Pre-built templates and automation recommendations

GET /api/automation/templates/
List all automation templates
POST /api/automation/recommendations/
Generate automation recommendations for a process

üìä Response Format

All endpoints return JSON responses. List endpoints support pagination with 'page' and 'page_size' parameters. Most endpoints support filtering, searching, and ordering.

üîç Query Parameters

Common query parameters include: search, ordering, page, page_size, and various filter fields specific to each endpoint.
